# 鲁班教学大师 · K12理科可视化Skill

> **项目定位**：将春秋末期顶尖工匠鲁班的科学思想与K12理科教育深度融合，通过交互式可视化让知识点"看得见、摸得着、用得上"。

---

## 一、项目愿景

### 1.1 核心价值主张

| 维度 | 描述 |
|------|------|
| **教育价值** | "知其然，更知其所以然"——每个知识点配真实场景，让学生理解原理而非死记结论 |
| **文化价值** | 鲁班代表中国古代应用科学的最高峰——数学(率)、力学(杠杆/滑轮)、机械学(连弩)、光学(透镜)，用他的视角重新解读K12理科 |
| **审美价值** | 墨韵黑金 + 榫卯暗纹 + 现代交互，既有文化厚度又有高级科技感 |

### 1.2 目标用户

- **一线教师**：备课参考、课堂演示
- **学生**：课后复习、主动探索
- **家长**：辅导孩子时的可视化工具
- **教育开发者**：基于此Skill二次开发

### 1.3 交付物清单

| 交付物 | 说明 | 路径 |
|--------|------|------|
| `luBan-teaching-skill` | GitHub开源Skill包，可被WorkBuddy调用 | `luBan-teaching-skill/` |
| `project-page` | 项目介绍落地页 | `project-page/index.html` |
| `case-studies` | 4个代表性知识点可视化案例 | `case-studies/` |
| `docs/` | 开发文档、API文档 | `docs/` |

## 四、四大可视化案例

> **选题原则**：选取具有"鲁班特色"的知识点——即有工程应用背景、有明确生活场景、能做交互实验的知识点。

### 案例1：杠杆原理（物理·八年级）

```
┌─────────────────────────────────────────────────────────────┐
│ 鲁班心法                                                      │
├─────────────────────────────────────────────────────────────┤

│                                                             │

│  · 剪刀(省力杠杆)：剪铁皮、修剪树枝                          │
│  · 筷子(费力杠杆)：夹细小食物需要精细控制                     │

┌─────────────────────────────────────────────────────────────┐
│ 鲁班开物                                                      │

┌─────────────────────────────────────────────────────────────┐
│ 鲁班心法                                                      │

│                                                             │
│ [实时显示]                                                    │

┌─────────────────────────────────────────────────────────────┐
│ 鲁班问答                                                      │
├─────────────────────────────────────────────────────────────┤
│ Q1. 光从空气垂直射入水中时，传播方向会怎样？                  │
│    A. 向法线偏折  B. 背向法线偏折  C. 不变  D. 无法确定       │
│    → 答案：C. 不变（垂直入射不偏折）                          │
│                                                             │

│    A. 40.8°  B. 45°  C. 22.5°  D. 60°                       │
│    → 答案：A. 40.8°（sin60°/1.33 ≈ 0.65， arcsin ≈ 40.8°）    │
│                                                             │

│                                                             │
│ Q4. 全反射现象发生在什么条件下？                              │
│    A. 光从光密介质到光疏介质，入射角>临界角                   │
│    B. 光从光疏介质到光密介质，入射角>临界角                   │
│    C. 任何情况下都可以发生  D. 光垂直入射时                    │
│    → 答案：A                                                 │
│                                                             │
│ Q5. 为什么钻石看起来特别闪亮？                               │
│    → 解析：钻石折射率2.42高，全反射临界角小，                 │
│          切割角度设计使光在内部多次全反射，产生火彩           │
└─────────────────────────────────────────────────────────────┘
```

### 案例3：二元一次方程组（数学·七年级）

```
┌─────────────────────────────────────────────────────────────┐
│ 鲁班心法                                                      │

│ 【理论】方程组解的几何意义：两条直线的交点坐标                 │

│  · 购物决策：哪家店更划算？                                  │

│    - 店B：基础费0元，每件5元                                 │
│    → 买5件时：A=25元，B=25元；买10件时：A=40元，B=50元       │
│                                                             │
│  · 行程规划：两人相向/同向而行，求相遇时间                   │
│  · 配料配比：两种原料混合达到目标浓度                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 鲁班开物                                                      │
├─────────────────────────────────────────────────────────────┤

│  · 直线2参数：斜率 m₂ (-5~5)，截距 b₂ (-10~10)               │
│  · 预设场景按钮：                                            │
│    "购物决策" / "行程相遇" / "配料配比"                      │
│  · 解题步骤显示：开关                                         │
│                                                             │
│ [实时显示]                                                    │
│  · 方程组标准形式：a₁x + b₁y = c₁                           │
│  · 交点坐标 (x, y)                                          │
│  · 解的类型：无解/唯一解/无数解                              │
│  · 坐标系动态绘图                                            │
│  · 解的验证（代入原方程检验）                                │

┌─────────────────────────────────────────────────────────────┐
│ 鲁班问答                                                      │
├─────────────────────────────────────────────────────────────┤

│    A. (3, 2)  B. (2, 3)  C. (4, 1)  D. (3, 1)               │
│    → 答案：A（相加得2x=6，x=3；y=2）                          │
│                                                             │
│ Q2. 若两条直线平行无交点，方程组的解的情况是？                │

│    → 答案：B（矛盾方程，无解）                               │
│                                                             │
│ Q3. 用代入法解 {2x + y = 7, x - y = 2}，第一步是？            │
│    A. 将 x = y + 2 代入第一个方程                            │
│    B. 将 y = 2x - 7 代入第二个方程                           │
│    C. 两式相加消元  D. 将 x = 2 + y 代入                     │
│    → 答案：A                                                 │
│                                                             │
│ Q4. "代入消元法"的核心思想是什么？                            │
│    → 解析：用一个问题表示另一个问题，                        │
│          将二元转化为一元——鲁班"以简驭繁"思想                │
│                                                             │
│ Q5. 方程 {ax + by = 6, 2ax + 2by = 12} 有多少解？           │
│    A. 0个  B. 1个  C. 无数个  D. 无法确定                    │
│    → 答案：C（两方程等价，无数解）                           │
└─────────────────────────────────────────────────────────────┘
```

## 六、教育准确性保证方案

### 6.1 核心原则

```
┌─────────────────────────────────────────────────────────────┐
│                    教育准确性等级                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  L1 【精确级】 公式/数值/定律 —— 必须与权威来源100%一致       │
│     例：水的折射率 n=1.33，光速 c=299792458 m/s              │
│     → 数据来源：NIST CODATA / 人教版教材                     │
│                                                             │
│  L2 【合理级】 场景/比喻/应用 —— 符合科学原理，表述需审核     │
│     例：筷子折断"看起来"变弯，而非"真的"变弯                  │
│     → 需教研团队审核表述准确性                                │
│                                                             │
│  L3 【开放级】 开放性问题/思考题 —— 允许多种合理解答          │
│     例："还有什么生活中的折射现象？"                         │
│     → 提供参考答案库，但不限制创新答案                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 验证流程

```
数据输入 → 专家库匹配 → 公式自动验算 → 场景合理性审核
    ↓           ↓             ↓              ↓
  来源标注   相似知识点    数学软件验证    教研团队审核
    ↓           ↓             ↓              ↓
  L1数据优先  交叉验证      Wolfram验证     用户反馈迭代
```

### 6.3 公式精度要求

| 类型 | 要求 | 实现方式 |
|------|------|----------|
| 物理常数 | 绝对精确 | 内置常量表（不依赖运行时计算）|
| 三角函数 | 浮点精度 | Math.js符号计算 |
| 指数/对数 | 保留5位有效数字 | toPrecision(5) |
| 角度 | 保留1位小数 | toFixed(1) + ° |
| 百分比 | 保留1位小数 | toFixed(1) + % |
