# PRD — 鲁班教学大师

## 作品目标

打造一个能展示K12理科可视化教育的落地页，突出中国古风+现代科技的设计语言，证明鲁班IP在教育可视化领域的独特价值。

## P0 主路径

```
首页（落地页）
  ├── Hero区域：大标题 + 鲁班IP介绍
  ├── 案例展示区：4个知识点卡片
  │     └── 点击卡片 → 进入案例详情页
  └── 底部：GitHub链接 + 技术栈说明

案例详情页（以杠杆原理为例）
  ├── 顶部导航：返回 + 知识点标题 + 全屏按钮 + 模式切换 + 三标签
  ├── 3D可视化画布（占主区域）
  ├── 三板块标签页：
  │     ├── 鲁班心法（理论+生活场景）
  │     ├── 鲁班开物（交互控件+实时参数）
  │     └── 鲁班问答（5道选择题）
  └── 底部信息栏：实时参数显示
```

## P1 Wow 点

- [ ] 3D杠杆物理实验，拖拽支点/砝码实时反映力矩变化
- [ ] 标签页右侧滑出动画（400ms cubic-bezier）
- [ ] Dark/Light模式一键切换
- [ ] 全屏模式下3D场景沉浸式展示

## P2 可选项

- [ ] 案例切换动画过渡
- [ ] 光线折射的完整光路追踪
- [ ] 元素周期表的3D球棍模型

## 非目标

- 不做完整Skill调用机制
- 不做用户登录/数据存储
- 不做移动端适配
