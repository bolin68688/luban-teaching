# UI/UX — 鲁班教学大师

## 视觉方向

**墨韵黑金 + 现代科技感**
- 深色背景（墨黑 #1A1A2E）配暖金色点缀（金铜 #C9A227）
- 榫卯暗纹作为装饰点缀（透明度5-10%）
- 衬线体标题（文化感）+ 无衬线正文（可读性）
- 卡片圆角8px，边框微光效果

**首屏设计**：
- 背景：深墨色 + 微弱粒子动效（代表分子/原子）
- 标题：站酷黄油体，金铜色渐变
- 副标题：思源宋体，中灰色

## 关键交互

### 三板块标签切换

```
初始状态：
┌─────────────────────────────┬──────────────────────────────┐
│                             │                              │
│    Three.js 可视化区域       │                              │
│    （占满右侧）              │                              │
│                             │                              │
│                             │                              │
├─────────────────────────────┴──────────────────────────────┤
│  [鲁班心法]  │  [鲁班开物]  │  [鲁班问答]                    │
└─────────────────────────────────────────────────────────────┘

点击标签后（右侧面板滑入）：
┌─────────────────────────────┬──────────────────────────────┐
│                             │   鲁班心法                    │
│    Three.js 可视化区域       │   ─────────                  │
│    （收缩至65%）             │   【理论】杠杆平衡...         │
│                             │                              │
│                             │   【鲁班视角】...            │
│                             │                              │
│                             │   【生活场景】...             │
└─────────────────────────────┴──────────────────────────────┘
```

**动效参数**：
- 面板宽度：35%
- 动画时长：400ms
- 缓动函数：cubic-bezier(0.22, 1, 0.36, 1)
- 面板背景：玄墨 #16213E + 榫卯暗纹

### 全屏模式

- 隐藏顶栏、底栏、标签栏
- Three.js画布全屏
- 右下角浮动"ESC退出"按钮
- 退出时还原所有UI
